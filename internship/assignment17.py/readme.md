# PySpark Sales DataFrame Application

## Objective
Perform DataFrame operations using PySpark.

## Dataset
sales.csv contains:
- product_id
- product_name
- category
- sales

## Operations Performed
1. Sort products by sales in descending order.
2. Display top 3 highest-selling products.
3. Filter products with sales greater than 80,000.
4. Save filtered records as CSV.

## Run Locally

```bash
python app.py
