# 🚀 PySpark Employee RDD Project

## 📌 Project Description
This project processes employee data using **PySpark RDDs** inside a Docker container.

---

## 📊 Features
- Sort employees by salary (descending)
- Calculate department-wise total salary
- Find top 3 highest-paid employees
- Save output to file
